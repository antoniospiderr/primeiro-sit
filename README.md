# Gerador de Curriculo Simples (PHP, HTML, CSS, JavaScript)
Este é um projeto de um gerador de curriculos
basico, desenvolvido ultilizando PHP para
prossesar dados, HTML para estrurar, CSS
para estilização e JavaScript para interatividade
(como pré-visualização ou validação básica).

O objetivo é permitir que o usuário insira suas
informações pessoais e profissionais através de
um formulário e,  em seguida, gere um curriculo
formatado pronto para impressão ou 
exportação (simulada).

# Funcionalidades

**Formulari de Entrada:** Interface HTML/CSS
para coletar dados do usuário (dados
pessoais, formação, experiência
etc...)

**Processamento PHP:** O script PHP recebe os dados via POST, valida-os (se necessário) e os formata em um layout de currículo.

**Estilização Profissional:** Uso de CSS para garantir que o currículo gerado tenha uma aparência limpa e profissional.

**Interatividade (JavaScript):** Possível uso de JavaScript para validação de formulário em tempo real, visualização prévia dinâmica antes da submissão ou funcionalidades de UI/UX.
Geração de Saída: O PHP renderiza a página final do currículo, que pode ser facilmente impressa.

 ## Tecnologias Utilizadas 
**Frontend:** HTML, CSS, JavaScript 

**Backend:** PHP
